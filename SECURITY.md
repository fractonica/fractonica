# Security policy

Fractonica is pre-release software and must not yet be exposed directly to an
untrusted network. The current node API deliberately refuses non-loopback bind
addresses until authentication and pairing are implemented.

Please report vulnerabilities privately through this repository's GitHub
Security Advisory form. Do not open a public issue containing exploit details,
private keys, access tokens, personal data, database contents, or media.

There are no supported production release lines yet. Once releases begin, this
document will list supported versions and disclosure timelines.
